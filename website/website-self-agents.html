Website Self‑Agents

Our website is the first customer of our Multi‑Agent System

We run lightweight agents on every change to keep the site fast, consistent and accessible. Results appear in the Status dashboard and as PR comments.

View live status See agents

Agents (MVP)

Layout Agent

Consistent spacing, grid and typography

  • Normalizes paddings/margins and heading hierarchy.
  • Checks buttons/links for empty labels and invalid hrefs.
  • Writes a Markdown report and (optionally) opens a PR.
Healthcheck Agent

Broken links & HTML quality

  • 404 & redirect checks across the site.
  • Validates basic HTML semantics and ARIA landmarks.
  • Updates the Status dashboard with findings.
Image Consistency Agent

Sizes, aspect ratio & alt text

  • Flags over‑wide images and missing alt attributes.
  • Suggests optimized widths for hero/poster images.
  • Exports a CSV/JSON with suggested fixes.
SEO Agent

Meta, Open Graph & sitemap

  • Ensures title/description length and uniqueness.
  • Generates Open Graph/Twitter tags from page content.
  • Keeps sitemap.xml in sync.

How it runs

Trigger

On every push to main (and nightly), GitHub Actions runs the agents inside the repo.

Outputs

Agents write reports to /logs/, post a Status entry, and may open an automated PR when a fix is safe.
