Gaia Principles

Roles, Permissions, Responsibilities, and Protocols define how agents collaborate: LeadAgent (GEORGE), SubLeadAgent (MES), and SpecializedAgents operate with clear accountability.

ISA‑95 Alignment

Agent Modeling Language (AML) — virtautoOEM 
system virtautoOEM {

  roles {
    LeadAgent role;             // GEORGE – Enterprise Orchestrator
    SubLeadAgent role;          // MES – Production Orchestrator
    StrategicAgent role;        // CEO, CFO, CTO, COO
    FunctionalAgent role;       // R&D, SupplyChain, Procurement, Quality, IT, HR, Finance, Sales
    OperativeAgent role;        // Planner, Logistics, Maintenance, QualityMgr, ProjectLead
    SupportAgent role;          // Legal, Compliance, PR, AfterSales, Sustainability
    CrisisAgent role;           // SWAT
    ExternalActor role;         // Customer, Supplier, Market, Regulator
  }

  agents {
    GEORGE : LeadAgent; MES : SubLeadAgent;
    CEO, CFO, CTO, COO : StrategicAgent;
    RnD, SupplyChain, Procurement, Finance, Quality, ITPlatform, HR, SalesMarketing : FunctionalAgent;
    Produktionsplaner, Logistik, Instandhaltung, Qualitätsmanager, Projektleiter, Robotersteuerung, PrototypValidator : OperativeAgent;
    Recht, Compliance, PRKrise, AfterSales, Nachhaltigkeit, RisikoAssessment, FacilityEnergie, DatenGovernance, SecurityCyber : SupportAgent;
    SWATKrise : CrisisAgent;
    Customer, Supplier, Market, Regulator : ExternalActor;
  }

  protocols {
    Negotiation(CFO, Procurement, Supplier);
    Coordination(COO, Produktionsplaner, SupplyChain, Logistik, MES);
    InformationSharing(Quality, AfterSales, RnD);
    ErrorHandling(SWATKrise, COO, SupplyChain, MES);
    DecisionMaking(GEORGE, CEO, CFO, CTO, COO, all FunctionalAgents);
    ComplianceCheck(Compliance, Recht, Procurement, CFO);
    CrisisCommunication(PRKrise, CEO, Quality, AfterSales);
  }

  responsibilities {
    GEORGE ensureGlobalAlignment(OKRs, KPIs); MES ensureProductionFlow();
    CEO defineVisionAndM&A(); CFO ensureBudgetControl(); CTO ensureInnovationRoadmap(); COO ensureOperationalExcellence();
    SupplyChain ensureMaterialFlow(); Procurement manageSuppliers(); Finance reportFinancials(); Quality ensureComplianceAndTesting();
    ITPlatform integrateSystems(); HR ensureResourceAllocation(); SalesMarketing captureMarketFeedback();
    Produktionsplaner scheduleProduction(); Logistik ensureTransportFlow(); Instandhaltung manageMaintenance();
    Robotersteuerung controlRobots(); PrototypValidator validatePrototypes(); Projektleiter manageProjects();
    Recht provideLegalSupport(); Compliance ensureGovernance(); PRKrise manageCommunications(); AfterSales manageCustomerService();
    Nachhaltigkeit ensureESGCompliance(); RisikoAssessment provideRiskAnalysis(); FacilityEnergie manageEnergy();
    DatenGovernance ensureDataQuality(); SecurityCyber ensureCyberSecurity(); SWATKrise solveCriticalIssues();
  }

  environment { Level0_PhysicalProcesses; Level1_ControlSystems; Level2_SupervisorySystems; Level3_MES; Level4_Enterprise; Level5_Strategic; }
  memory { ShortTerm: ProtocolBuffers; LongTerm: DataWarehouse, Historian, ERP integration; Strategic: OKR Repository, Knowledge Base; }
}
