Website Self-Agents

Lightweight, specialized helpers for our website maintenance: They ensure a consistent layout, clean images, and an ongoing health check of deployments.

Layout Agent

Keeps spacing, widths and grid consistent. Uses the common styles.css and enables unified Hero, Footer and Buttons.

  • Standardizes Spacing & Grid
  • Clears up duplicate sections
  • Desktop-First, responsive Breakpoints

Image Consistency Agent

NNormalized posters/illustrations: maximum width, consistent curves/shadows, clean retina display.

  • Uniform max width
  • Clean edges/radius
  • Alt texts & lazy loading

Healthcheck Agent

Monitors deployments and the Pages build, updates the status dashboard, and reports any anomalies.

  • CI/CD-signal check
  • Update status list
  • Rollback-Trigger ready
